Introduction

　

Qingkai (Thomas) Shi is now a PhD student in the Department of Computer Sci. & Eng., the Hong Kong University of Science and Technology (HKUST). His advisor is Prof. Charles Zhang. His research interest is system security, program analysis and testing. Before joining HKUST in Sep. 2015, he got a B. Projects

  • Pinpoint

    Industrial-strength next-generation automated bug finding tools through static analysis and AI. It is implemented for C/C++ programs based on LLVM. It has found more than 40 vulnerabilities in about a dozen of open-source projects, including Apache, MySQL, Firefox, Python, OpenSSL, etc. This project has been commercialized at Sourcebrella Inc..

  • Canary

    Canary is a set of tools built on a unification-based alias analysis. Currently, it contains an order-based record/replay tool, and a trace-based bug detector for concurrent C/C++ programs. It is implemented for C/C++ programs based on LLVM. This project currently is applied by the Sourcebrella Inc. for vulnerability detection.

  • Swan

    Swan is a prototype tool for verifying whether an atomicity violation is fixed sufficiently or not. It analyzes a dynamic buggy trace with fix (i.e., synchronization) information. It does not require developers should fully understand the bugs before fixing, thus being more practical. It is implemented for Java programs based on Soot. The technical paper has been accepted by TSE.
